[orange]与 Siri 一起体验旅途！[normal]Apple Siri 语音导航将 Apple 设备上标志性的声音带到卡车模拟游戏中。有 [orange]12 种语言的 70 多种声音[normal]可供选择，Siri 在这里为你的货运冒险提供指引，让你的旅程更加愉快和难忘。

[orange]2025 年 06 月 30 日[normal]: 2.0 ＞ [green]3.0[normal]。


[orange]介绍[normal]
我是一个 17 岁的学生，这个项目是我第一次尝试创造这样的东西。它是用来学习和练习的。
更多的语言和声音将在未来添加。如果你说的语言或者你喜欢的声音还不可用，请随意地[orange]留下你的请求[normal]。你的建议可能会在短短几天内实现！
感谢你使用这个模组！如果你喜欢它，请考虑在创意工坊投票或在 GitHub 留下一颗星、分享你的反馈、用你的免费 Steam 点数奖励这个项目，或者 buy me a coffee 来支持我的“工作”！
要帮助改进这个项目，你可以在 GitHub 上的存储库 at [orange]https://github.com/quinn0823/apple-siri-voice-navigation[normal] 查看开发指南。你也可以在那里查看使用其他语言的描述。

[orange]兼容性[normal]
[blue]1.43+[normal]，同时支持 ETS2 和 ATS。

[orange]使用[normal]
1. 在标题界面点击[orange]模组[normal]来打开模组管理器。(你已经在这里了！)
2. 双击 [orange]Apple Siri Voice Navigation[normal] 来启用这个模组。
3. 打开[orange]选项[normal]。
4. 导航到[orange]音频 ＞ 语音导航 ＞ 语言和声音[normal]并选择一个你喜欢的声音。
